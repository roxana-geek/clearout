# 📦 Продай пока спишь — Инструкция по запуску

Веб-приложение: фотографируешь вещь → AI генерирует объявления на иврите, русском и английском.

---

## 🚀 Деплой на Vercel — шаг за шагом (10–15 минут)

### Шаг 1 — Получи API ключ Anthropic (бесплатно)

1. Открой [console.anthropic.com](https://console.anthropic.com)
2. Зарегистрируйся (Google аккаунт подойдёт)
3. Перейди в **API Keys** → **Create Key**
4. Скопируй ключ — он выглядит так: `sk-ant-api03-...`
5. Сохрани его — он покажется только один раз!

> 💡 Новые аккаунты получают $5 бесплатных кредитов. Одно объявление стоит ~$0.01. Хватит на ~500 объявлений бесплатно.

---

### Шаг 2 — Загрузи код на GitHub

1. Открой [github.com](https://github.com) → войди или зарегистрируйся
2. Нажми **New repository** (зелёная кнопка)
3. Название: `clearout`
4. Нажми **Create repository**
5. На следующей странице нажми **uploading an existing file**
6. Перетащи ВСЕ файлы из папки `clearout` (включая папки `src` и `public`)
7. Нажми **Commit changes**

---

### Шаг 3 — Задеплой на Vercel

1. Открой [vercel.com](https://vercel.com)
2. Нажми **Sign Up** → войди через GitHub
3. Нажми **Add New Project**
4. Выбери репозиторий `clearout` → нажми **Import**
5. В разделе **Environment Variables** добавь:
   - **Name:** `REACT_APP_ANTHROPIC_KEY`
   - **Value:** вставь свой ключ `sk-ant-api03-...`
6. Нажми **Deploy**
7. Через 2–3 минуты получишь ссылку вида `clearout-xyz.vercel.app` ✅

---

### Шаг 4 — Добавь на телефон как приложение

**На iPhone (Safari):**
1. Открой свою ссылку в Safari
2. Нажми кнопку "Поделиться" (квадрат со стрелкой вверх)
3. Прокрути вниз → **На экран "Домой"**
4. Нажми **Добавить**

**На Android (Chrome):**
1. Открой ссылку в Chrome
2. Нажми три точки → **Добавить на главный экран**
3. Нажми **Добавить**

Теперь на экране появится иконка — приложение открывается как обычное приложение! 📱

---

## 📱 Как пользоваться

1. Открой приложение
2. Нажми на область с 📦 — откроется камера или галерея
3. Сфотографируй вещь или выбери фото
4. Укажи желаемую цену и минимальную цену
5. Добавь заметки (необязательно)
6. Нажми **✨ Создать объявления**
7. Через 15–20 секунд получишь 3 готовых объявления
8. Нажми кнопку нужной платформы — откроется Telegram/Facebook или скопируется текст для Yad2

## 🛠️ Если что-то не работает

**Ошибка "API key"** → проверь что ключ добавлен в Vercel Environment Variables

**Фото не загружается** → попробуй меньший размер фото, или конвертируй HEIC в JPG

**Объявления не генерируются** → проверь баланс на console.anthropic.com
